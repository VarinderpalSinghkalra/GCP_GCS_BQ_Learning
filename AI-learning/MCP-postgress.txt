## 4. Inventory Table Schema

A production-ready `inventory` table was created with proper constraints:

```sql
CREATE TABLE inventory (
    id SERIAL PRIMARY KEY,
    item_id TEXT NOT NULL UNIQUE,
    item_name TEXT NOT NULL,
    quantity INT NOT NULL CHECK (quantity >= 0),
    location TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Key Design Notes

* `item_id` is **unique** and agent-friendly
* `quantity` enforces non-negative stock
* `location` supports multi-warehouse demos
* timestamps enable enterprise realism
